# Fashion Trend Prediction (Trend Catcher)

This project uses a deep learning model to classify fashion styles from outfit images and displays the results through a Streamlit web application.

This project was developed as part of a group assignment for a Data Science class.

## Tools
- Python
- PyTorch
- Streamlit

## Features
- Upload image for prediction
- Classify into 5 styles (Athleisure, Fairycore, Old Money, Streetwear, Y2K)
- Show confidence score

## How to Run
pip install -r requirements.txt

streamlit run app.py



## Note
The trained model file (.pth) is not included.  
This project demonstrates the machine learning workflow and application interface.
